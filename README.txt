PROYECTO BUSCAMINAS - PYTHON POO

INSTRUCCIONES DE USO:
1. Clonar o copiar todo el proyecto en una carpeta.
2. Asegurarse de tener el archivo 'records.json' con los parámetros iniciales.
3. Ejecutar el archivo principal con:
   python main.py
4. El juego se ejecutará por consola.

FUNCIONALIDADES:
- Juego de Buscaminas con CLI
- Marcar casillas con bandera o signo de interrogación
- Expansión automática de casillas vacías
- Registro de los 3 mejores tiempos
- Lectura de configuración desde archivo
